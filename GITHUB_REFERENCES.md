## 💡 使用建议

### MVP 阶段（先实现）
1. 参考 `Freqtrade` 的交易所 API 集成
2. 使用 `Lightweight Charts` 做 K线图
3. 使用 `TA-Lib` 或 `Pandas-TA` 计算技术指标
4. 参考 `Stock Prediction Using LSTM` 做基础预测

### 增强阶段
1. 参考 `Deep Learning for Stock Prediction` 添加更多模型
2. 参考 `OrderFlow Analysis Tools` 做订单流分析
3. 参考 `FinBERT` 做情感分析

### 高级阶段
1. 参考 `FinRL` 做强化学习
2. 参考 `OctoBot` 做完整系统
