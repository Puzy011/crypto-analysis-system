# 虚拟货币行情分析系统 - 需求文档

## 📋 项目概述

设计一个功能完善的虚拟货币（加密货币）行情分析系统，提供实时行情、历史数据分析、技术指标、价格预警等功能。

---

## 🎯 核心功能需求

### 1. 实时行情模块

#### 1.1 行情数据展示
- **支持的交易所**：
  - Binance（币安）
  - OKX
  - Coinbase
  - Huobi（火币）
  
- **支持的交易对**：
  - BTC/USDT
  - ETH/USDT
  - 其他主流币种（可配置）

- **实时数据**：
  - 最新价格
  - 24h 涨跌幅
  - 24h 成交量
  - 24h 最高价/最低价
  - 买一/卖一价格
  - 实时盘口数据

#### 1.2 行情更新机制
- WebSocket 实时推送（优先）
- REST API 轮询（备选）
- 数据更新频率：1-5 秒可配置

---

### 2. K线图与历史数据模块

#### 2.1 K线数据
- **时间周期**：
  - 1分钟、5分钟、15分钟、30分钟
  - 1小时、4小时、1天、1周、1月
  
- **K线数据字段**：
  - 开盘价（Open）
  - 最高价（High）
  - 最低价（Low）
  - 收盘价（Close）
  - 成交量（Volume）

#### 2.2 历史数据
- 支持获取最多 2 年的历史数据
- 数据本地缓存机制
- 历史数据导出功能（CSV/Excel）

---

### 3. 技术指标分析模块

#### 3.1 趋势指标
- **MA（移动平均线）**
  - SMA（简单移动平均）
  - EMA（指数移动平均）
  - 周期可配置（5、10、20、60、120、200）
  
- **MACD（指数平滑异同移动平均线）**
  - DIF、DEA、MACD 柱状图
  
- **BOLL（布林带）**
  - 上轨、中轨、下轨

#### 3.2 动量指标
- **RSI（相对强弱指标）**
  - 周期 14
  - 超买超卖区域标记
  
- **KDJ（随机指标）**
  - K、D、J 三线
  
- **WR（威廉指标）**

#### 3.3 成交量指标
- **VOL（成交量）**
- **OBV（能量潮）**

#### 3.4 其他指标
- **ATR（平均真实波幅）**
- **CCI（商品通道指标）**

---

### 4. 价格预警模块

#### 4.1 预警类型
- **价格突破预警**
  - 价格突破某个价位
  
- **涨跌幅预警**
  - 24h 涨跌幅超过阈值
  
- **技术指标预警**
  - RSI 超买/超卖
  - MACD 金叉/死叉
  - KDJ 金叉/死叉

#### 4.2 预警方式
- 系统弹窗
- 声音提醒
- 邮件通知
- Telegram/飞书机器人通知

#### 4.3 预警管理
- 添加/编辑/删除预警
- 预警历史记录
- 预警开关控制

---

### 5. 自选币模块

#### 5.1 自选管理
- 添加/删除自选币
- 自选币排序
- 自选币分组（投资组合、观察等）

#### 5.2 自选页面
- 快速查看自选币行情
- 批量更新价格
- 投资组合盈亏计算

---

### 6. 策略回测模块

#### 6.1 回测功能
- 支持自定义策略
- 基于历史数据回测
- 回测报告生成

#### 6.2 策略指标
- 总收益率
- 年化收益率
- 最大回撤
- 夏普比率
- 胜率
- 盈亏比

---

### 7. 数据统计与报表模块

#### 7.1 市场概览
- 币种总市值排名
- 24h 涨跌幅排行
- 24h 成交量排行
- 市场恐慌贪婪指数

#### 7.2 个人报表
- 投资收益报表
- 交易记录分析
- 资产配置分析

---

## 🏗️ 非功能需求

### 1. 性能需求
- 页面加载时间 < 2秒
- 行情更新延迟 < 1秒
- 支持同时监控 50+ 币种
- K线图渲染流畅

### 2. 可用性需求
- 界面简洁直观
- 支持深色/浅色主题
- 响应式设计（支持桌面/平板/手机）
- 中文/英文多语言支持

### 3. 可靠性需求
- 数据采集失败自动重试
- 网络异常自动重连
- 异常情况友好提示
- 数据本地备份

### 4. 安全性需求
- API Key 加密存储
- 不保存用户交易密钥
- 数据传输加密（HTTPS/WSS）
- 不涉及真实交易（仅分析）

## 🔧 技术栈建议

### 前端方案
- **框架**: React / Vue 3
- **图表库**: TradingView Charting Library / ECharts / Lightweight Charts
- **UI 组件**: Ant Design / Element Plus
- **状态管理**: Redux / Pinia
- **HTTP 客户端**: Axios
- **WebSocket**: 原生 WebSocket / Socket.io

### 后端方案（可选，如需）
- **语言**: Python / Node.js / Go
- **框架**: FastAPI / Express / Gin
- **数据库**: SQLite（本地）/ PostgreSQL（服务端）
- **缓存**: Redis
- **任务队列**: Celery（Python）/ Bull（Node.js）

### 纯前端方案（推荐）
- 无需后端
- 数据直接从交易所 API 获取
- 本地存储使用 IndexedDB / localStorage
